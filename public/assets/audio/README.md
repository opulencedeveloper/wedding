# Background Music

Please add a voiceless (instrumental) wedding or love-themed music file here.

## File Requirements:
- Format: MP3 (recommended) or OGG
- Name: `wedding-music.mp3`
- Duration: 2-5 minutes (will loop automatically)
- Style: Instrumental, romantic, wedding-appropriate
- Volume: The component sets volume to 30% automatically

## Suggested Sources:
- Royalty-free music from:
  - Pixabay (https://pixabay.com/music/)
  - Free Music Archive (https://freemusicarchive.org/)
  - YouTube Audio Library (https://www.youtube.com/audiolibrary)
  - Incompetech (https://incompetech.com/music/royalty-free/)

## Search Terms:
- "wedding instrumental"
- "romantic piano"
- "love song instrumental"
- "wedding background music"
- "romantic classical"

## Example Files:
You can download a free instrumental wedding music file and save it as:
`public/assets/audio/wedding-music.mp3`

The music will automatically play when the user interacts with the page (click, touch, or keyboard) and will loop continuously.
